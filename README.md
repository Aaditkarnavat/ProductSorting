# ProductSorting

Python implementation of sorting algorithms using a Product data model.

## Description

This project explores sorting algorithms applied to a simulated product inventory. Products are defined by attributes such as price, popularity, and rating, and sorted using multiple algorithmic approaches.

## Contents

- **Product data class** – `id`, `name`, `price`, `popularity`, `rating`, `original_index`
- **Sorting implementations** – multiple algorithms compared for correctness and performance

## Technologies

- **Language:** Python
- **Libraries:** `dataclasses`, `random`

## Purpose

Written as part of Data Structures and Algorithms practice. Useful for reviewing sorting logic and Python object handling.
